24   * 	</li>
25   * 	<li><b>getUrl *</b>: URL used to
26   * 		<a href="http://data.bioontology.org/documentation#Class">get</a>
27   * 		classes (concepts) from BioPortal. It <i>must</i> include the following
28   * 		placeholders to be replaced with values for the query parameters:</li>
29   * 		<ul>
30   * 			<li>{0}: ontology name: the name of an ontology, as enumerated in

32   * 			<li>{1}: class id: the URI of the searched class (concept).</li>
33   * 		    <li>{2}: apiKey</li>
34   * 		</ul>
35   * 	<li><b>useProxy</b>: (not used) use proxy to connect to BioPortal? Defaults

37   * 	<li><b>timeout</b>: timeout in milliseconds for the connections to
38   * 		BiopPortal.</li>
39   * </ul>
40   * * Please note that both <code>searchUrl</code> and <code>getUrl</code> should
41   * include a parameter <code>apikey</code>, obtained after registering to the
