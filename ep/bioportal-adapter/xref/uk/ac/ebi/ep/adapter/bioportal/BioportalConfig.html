3   /**
4    * Configuration object for

6    * proxies. This bean should be used as a singleton.
7    * <br>
8    * Configurable properties (via JMX):
9    * <ul>
10   *  <li><b>apiKey *</b>: the API key <b>required</b> to make requests to the
11   *      BioPortalweb service.</li>
12   * 	<li><b>searchUrl *</b>: URL used to
13   * 		<a href="http://rest.bioontology.org/bioportal/search">search</a>
14   * 		BioPortal. It <i>must</i> include the following placeholders to be
15   * 		replaced with values for the query parameters:
16   * 		<ul>
17   * 			<li>{0}: ontologyids: a comma-separated list of ontology IDs (see
18   * 		        {@link BioportalOntology#getId()})</li>
19   * 			<li>{1}: query</li>
20   * 		    <li>{2}: apiKey</li>
21   * 		</ul>
22   * 		If any of these placeholders are missing, the URL should have them
23   * 		hardcoded.<br/>
24   * 	    The URL <i>can</i> also include these optional placeholders:
25   * 	    <ul>
26   * 			<li>{3}: isexactmatch (<code>1 | true</code> for true,
27   * 		    	<code>0 | false</code> for false)</li>
28   * 	    </ul>
29   * 	</li>
30   * 	<li><b>getUrl *</b>: URL used to
31   * 		<a href="http://rest.bioontology.org/bioportal/concepts">get</a>
32   * 		concepts from BioPortal. It <i>must</i> include the following
33   * 		placeholders to be replaced with values for the query parameters:</li>
34   * 		<ul>
35   * 			<li>{0}: ontologyversionid: the ID of a concrete version of the
36   * 		        ontology.</li>
37   * 			<li>{1}: conceptid: the ID of the searched concept.</li>
38   * 		    <li>{2}: apiKey</li>
39   * 		</ul>
40   * 	<li><b>useProxy</b>: (not used) use proxy to connect to BioPortal? Defaults
41   * 		to <code>true</code>.</li>
42   * 	<li><b>timeout</b>: timeout in milliseconds for the connections to
43   * 		BiopPortal.</li>
44   * </ul>
45   * * Please note that both <code>searchUrl</code> and <code>getUrl</code> should
46   * include a parameter <code>apikey</code>, obtained after registering to the
47   * service (free).
